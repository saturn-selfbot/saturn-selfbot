## 🐧 Linux
- Installation
  ```
  git clone https://github.com/saturn-selfbot/saturn-selfbot
  cd saturn-selfbot
  python -m pip install -r requirements.txt
  ```
- startup
  ```
  cd saturn-selfbot
  python3 -m saturn
  ```
